Remaining Differences From LLaVA-1.5

AxisCurrent status
LM size and initializationOfficial LLaVA-1.5 7B target uses Vicuna-v1.5-7B. Our reproduction runs use Gemma3-1B-IT, so the language model is much smaller and instruction-tuned under a different tokenizer/chat distribution.
Vision towerBoth use CLIP-L/14 at 336px in spirit. Our implementation uses OpenAI CLIP-L/14-336 with feature layer -2, patch features, MLP2x GELU projector, and JAX/HSDP. Image preprocessing is not an exact official-code reproduction because we use stretch resize.
Stage-1 dataOfficial target is LLaVA-Pretrain/LCS-558K. The first run used blip3o-short; the second run uses cc12m. Both are replacement data, not exact data-level reproduction.
Stage-2 sample definitionOfficial LLaVA-1.5 reports a 665K SFT recipe. Our training also uses 5195 x 128 = 664,960 stage-2 samples, but samples from a much larger LLaVA-OV1.5 pool plus weighted academic datasets. It is recipe-scale-aligned, not exact sample-list-aligned.
First-run missing SFT categoriesThe first run lacked OKVQA, A-OKVQA, OCRVQA, Visual Genome detection/grounded boxes, RefCOCO grounding, and ShareGPT text-only.
Current remaining data gapsThe second run adds OKVQA/A-OKVQA/OCRVQA/VG detection/RefCOCO, but still does not use exact LLaVA-1.5 665K JSON/conversations and still omits ShareGPT text-only.
Extra/different dataLLaVA-OV1.5-Instruct is newer and much larger than LLaVA-1.5 665K. This may improve breadth but makes the result an ablation-style reproduction rather than a strict LLaVA-1.5 data reproduction.
Eval protocolWe track several official metrics. POPE is aligned to the adversarial split in the spreadsheet: final logs contain both macro F1 and adversarial F1, and the first and second runs record the adversarial F1. We do not yet track MMBench-CN, SEED all/video, LLaVA-Wild, or MM-Vet in the spreadsheet. We additionally track VStar/OCRBench/MMVP/CountBench/RefCOCOg/KNN, which are not original LLaVA-1.5 paper-table metrics.

Main Gaps

  • MME-P: second run is 1064, target 1511, delta -447.
  • VisWiz: second run is 13.89, target 50, delta -36.11. This is still the largest paper-metric percentage gap, even after the +8.47 first-run-to-second-run gain.
  • TextVQA: second run is 33.82, target 58.2, delta -24.38. OCRVQA did not move this metric yet.
  • VQAv2: second run is 54.86, target 78.5, delta -23.64, but it improved strongly from the first run.
  • MMBench: second run is 42.44, target 64.3, delta -21.86.
  • MMVP remains above the spreadsheet target by +27.63, but it is not an original LLaVA-1.5 paper metric and should be protocol-checked before treating it as a reliable win.
  • POPE is close to the spreadsheet target: the second run records adversarial F1 80.05 versus target 84.2.
